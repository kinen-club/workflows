# Kinen Space

This workspace uses the kinen methodology for structured thinking sessions.

**Full methodology:** https://kinen.club/methodology.md

## Setup

Install the kinen plugin:

```bash
claude plugins add kinen-club/agents/claude-code
```

## Commands

| Command | Description |
|---------|-------------|
| `/session` | Start a new kinen session |
| `/round` | Continue to the next round |
| `/summarize` | Create session summary |
